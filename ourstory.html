About Us

Welcome to the Recipe Room, our project for the Code First Intro to Web Development Course! We decided to create a recipe website to showcase several easy recipes for each meal.


How we did it

Firstly, we decided on a layout for our website and created wireframes based on our ideas.



We decided on a green background as this colour is associated with health and quality, which we believe are good things to promote through our recipe website. To make the headings stand out, we chose a light pink colour that contrasts nicely with our dark green background. White was an obvious choice for the text as it is important that the user can read it, this is the same reason we chose a simple font.


For a css framework, we used bootstrap. To customise the bootstrap features we used, we overrode it with our own style sheet.


Extra Features


If we had more time we would've liked to add the following features:

  • Our own individual pages for the recipes, rather than external links
  • A search bar feature
  • Centre the navbar
  • Explore more bootstrap features
